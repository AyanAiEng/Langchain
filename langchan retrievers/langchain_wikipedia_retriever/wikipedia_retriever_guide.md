# 🧠 Wikipedia Retriever – Complete Guide

## What is a Wikipedia Retriever?

A Wikipedia Retriever is a tool used in AI systems (like LangChain) that automatically searches Wikipedia and returns relevant, structured information for use in language models.

Instead of relying only on training data, the model can fetch real-time factual knowledge from Wikipedia.

---

## ⚙️ How it works

1. User asks a question
2. Query is sent to Wikipedia API
3. Relevant articles are retrieved
4. Text is extracted and cleaned
5. Data is passed to the LLM (AI model)

This is part of Retrieval-Augmented Generation (RAG).

---

## 🏗️ Simple Example

User: "What is photosynthesis?"

System:
- Searches Wikipedia
- Finds article
- Extracts summary
- Sends it to AI model
- AI generates final answer
